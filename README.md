# Infra Learning

这个仓库记录在学习 AI infra 过程中遇到的一些算法、机制和工程概念的简单实现。

内容不一定都严格属于 infra，是把阅读论文、看源码或做实验时碰到的有意思想法，用尽量小的代码复现出来，方便理解和回顾。

## Contents

- `Speculative_Decoding`: speculative decoding 和 EAGLE 风格解码的学习实现
- `MoE_Learning`: Mixture of Experts 的核心 routing / expert / combine 逻辑

## Usage

```bash
pip install -e ".[dev]"
```
